# Interaction Design

This skill specializes in creating engaging and intuitive user interactions. It covers:

- User flow design and optimization
- Microinteractions and animations
- Interactive prototyping techniques
- User feedback and response design
- Gesture design for touch interfaces
- Voice and conversational interfaces
- Accessibility in interaction design
- Usability testing for interactions

Use this skill when designing interactive experiences that are intuitive, engaging, and user-friendly.
