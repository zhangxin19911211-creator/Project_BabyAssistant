# Frontend Design

This skill focuses on frontend design and implementation techniques. It covers:

- HTML5 and semantic markup
- CSS3 and modern styling techniques
- Responsive design and breakpoints
- CSS frameworks and preprocessors
- Frontend performance optimization
- Cross-browser compatibility
- Accessibility in frontend development
- Modern frontend tooling and workflows

Use this skill when working on frontend projects that require clean, responsive, and performant design implementation.
