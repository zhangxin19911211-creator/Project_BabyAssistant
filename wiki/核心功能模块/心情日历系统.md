# 心情日历系统

## 功能概述
心情日历是 V3 版本新增的核心功能，用于记录和展示宝宝的每日心情评价。支持多种心情评分、备注记录、日历可视化展示。

## 核心功能

### 1. 心情评价
- **评分类型**：很乖😊、普通😐、闹腾😫、生病🤒
- **评价规则**：
  - 每个宝宝每天只能有一条心情记录
  - 心情评分可覆盖更新
  - 备注可叠加添加（多条备注）
  - 不能评价未来日期

### 2. 日历视图
- 自研日历组件，支持 2024-2028 年范围
- 日历格子显示心情 emoji
- 支持月份切换、日期选择
- 显示宝宝头像和今日心情标记

### 3. 备注管理
- 支持文字备注（最多 200 字）
- 支持语音转文字（微信同声传译插件）
- 备注条目独立删除
- 记录人和时间戳

### 4. 活动日志
- 记录所有心情相关操作
- 支持分页加载
- 区分评分变更和备注添加

## 权限控制

| 角色 | 查看心情 | 添加/修改评分 | 添加备注 | 删除心情 |
|------|---------|--------------|---------|---------|
| 掌门人 (guardian) | ✅ | ✅ | ✅ | ✅ |
| 一级助理 (guardian) | ✅ | ✅ | ✅ | ✅ |
| 二级助理 (caretaker) | ✅ | ✅ | ✅ | ❌ |
| 围观 (viewer) | ✅ | ❌ | ❌ | ❌ |

## 数据模型

### moods 集合
```javascript
{
  _id: String,          // 记录ID
  babyId: String,       // 宝宝ID
  date: String,         // 日期 "YYYY-MM-DD"
  rating: String,       // 评分 good/normal/naughty/sick
  emoji: String,        // 表情符号
  notes: [{             // 备注数组
    content: String,    // 备注内容
    openid: String,     // 记录人
    nickName: String,   // 记录人昵称
    createTime: Date    // 创建时间
  }],
  createTime: Date,     // 创建时间
  updateTime: Date      // 更新时间
}
```

### activityLogs 集合
```javascript
{
  _id: String,          // 日志ID
  familyId: String,     // 家庭ID
  babyId: String,       // 宝宝ID（可选）
  type: String,         // 操作类型
  description: String,  // 描述
  operatorOpenid: String, // 操作人
  createTime: Date      // 创建时间
}
```

## API 接口

### getMoodsByMonth
获取某月心情记录
```javascript
api.getMoodsByMonth(babyId, year, month)
```

### getMoodByDate
获取某天心情详情
```javascript
api.getMoodByDate(babyId, date)
```

### addMood
添加/更新心情评价
```javascript
api.addMood({
  babyId: String,
  date: String,
  rating: String,
  note: String  // 可选
})
```

### deleteMood
删除心情记录
```javascript
api.deleteMood(moodId)
```

### deleteMoodNote
删除单条备注
```javascript
api.deleteMoodNote(moodId, entryId)
```

## 性能优化

1. **并行加载**：使用 `Promise.all` 并行获取家庭和宝宝列表
2. **缓存策略**：家庭和宝宝列表缓存 5 分钟
3. **分页加载**：活动日志支持分页，每页 20 条
4. **按需渲染**：日历仅渲染当前月份数据

## 注意事项

1. 语音转文字依赖微信同声传译插件，需在 `app.json` 中声明
2. 心情评分可被覆盖，备注可叠加
3. 删除心情需要一级助理及以上权限
4. 日历日期计算已验证 2024-2028 年正确性
