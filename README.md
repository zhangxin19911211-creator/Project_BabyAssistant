# 宝宝助手 - 微信小程序

一款专为家庭设计的宝宝成长记录工具，支持多成员协作，帮助您轻松追踪宝宝的成长数据。

## 📱 功能特性

### 核心功能
- **家庭管理**：创建家庭、加入家庭、退出家庭、修改家庭名称
- **宝宝信息管理**：添加、查看、修改、删除宝宝信息（每个家庭最多3个宝宝）
- **成长记录追踪**：记录宝宝的身高、体重等数据
- **成长曲线分析**：通过图表直观展示宝宝的成长趋势，区分性别使用国家卫健委标准
- **多成员协作**：支持邀请家庭成员，分配不同权限角色
- **微信一键登录**：使用微信账号快速登录
- **云端数据存储**：数据安全存储在腾讯云

### 权限系统
- **监护人**：最高权限，可添加/删除宝宝、管理成员权限、邀请成员
- **照看者**：可添加成长记录，无法添加/删除宝宝或调整权限
- **围观者**：只能查看宝宝数据，无法进行其他操作

### 技术特点
- **微信原生开发**：使用小程序原生框架
- **腾讯云开发**：无需后端服务器，快速部署
- **数据安全**：用户数据隔离，权限验证
- **响应式设计**：适配各种手机屏幕
- **性能优化**：代码精简，加载迅速
- **色彩系统**：每个家庭有独特颜色，宝宝卡片颜色与家庭一致

## 🛠️ 技术栈

- **前端**：微信小程序原生框架 (WXML, WXSS, JavaScript)
- **后端**：腾讯云开发 (CloudBase)
- **数据库**：云数据库 (NoSQL)
- **图表**：ECharts 小程序版
- **认证**：微信登录

## 🚀 快速开始

### 环境要求
- 微信开发者工具
- 腾讯云开发环境
- Node.js (用于云函数开发)

### 部署步骤

1. **克隆项目**
   ```bash
   git clone <仓库地址>
   cd BabyAssistant_wechat_1
   ```

2. **配置云开发环境**
   - 在微信开发者工具中打开项目
   - 点击「云开发」按钮，创建或选择云开发环境
   - 复制环境ID到 `miniprogram/app.js` 中的 `globalData.env`

3. **部署云函数**
   - 在微信开发者工具中，右键点击 `cloudfunctions/login` 文件夹
   - 选择「上传并部署」

4. **创建数据库集合**
   - 在云开发控制台中，创建以下集合：
     - `babies`：存储宝宝信息
     - `records`：存储成长记录
     - `users`：存储用户信息
     - `families`：存储家庭信息
     - `inviteCodes`：存储邀请码

5. **运行项目**
   - 在微信开发者工具中点击「编译」按钮
   - 可以在模拟器中预览，或使用真机调试

## 📖 使用说明

### 1. 家庭管理
- **创建家庭**：首次使用时创建家庭，成为创建者（默认监护人）
- **加入家庭**：通过邀请码加入其他家庭
- **邀请成员**：监护人或照看者可以邀请其他成员加入家庭
- **修改家庭名称**：创建者可以修改家庭名称（最多7个字符）
- **退出家庭**：创建者退出会删除整个家庭，其他成员退出不影响家庭

### 2. 添加宝宝
- 进入首页，点击「添加宝宝」按钮
- 选择宝宝所属家庭
- 填写宝宝姓名（最多7个字符）、性别、出生日期等信息
- 可以选择上传宝宝头像
- 点击「完成」按钮保存

### 3. 记录成长数据
- 在宝宝详情页，点击「添加记录」按钮
- 填写测量日期、身高、体重等数据
- 点击「保存记录」按钮

### 4. 查看成长曲线
- 在宝宝详情页，查看身高和体重的成长曲线
- 系统会根据宝宝性别使用相应的国家卫健委标准数据
- 可以缩放查看不同时期的成长趋势
- 标准曲线（P3、P50、P97）和宝宝数据对比展示

### 5. 管理宝宝信息
- **修改宝宝姓名**：监护人可以点击宝宝姓名进行修改
- **删除宝宝**：在首页左滑宝宝卡片，可以删除宝宝（仅监护人）
- **查看详情**：点击宝宝卡片进入详情页

### 6. 成员管理
- **修改用户名**：每个成员可以修改自己的用户名（最多7个字符）
- **调整权限**：监护人可以调整其他成员的权限角色
- **移除成员**：监护人可以移除其他成员

## 🔒 数据安全

- **用户隔离**：每个用户只能查看和管理自己有权限的家庭和宝宝信息
- **权限验证**：所有数据操作都经过权限验证
- **事务保护**：删除操作使用事务确保数据一致性
- **登录安全**：使用微信官方登录机制
- **邀请码安全**：邀请码12小时过期，使用后自动删除
